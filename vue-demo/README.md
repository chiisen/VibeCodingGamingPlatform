# Vue DEMO Project

This is a DEMO project based on the analysis of `PROMPT.mt`.

## Setup

```bash
npm install
npm run dev
```

## Structure

- `src/components`: Vue components.
- `src/data`: Game data.

## Note on Icons
Game icons are currently placeholders as the scraping tool could not retrieve the original images automatically. You can update `src/data/games.json` with real URLs.
